# 3D-playground

## usage

* download or clone the project
* $ npm i
* $ npm run dev
* open localhost:8080 on your web browser (Tested on Chrome, Firefox, Microsoft Edge)

## development environment

1. Programming Languages

* Javascipt & Typesciprt (ES5)

2. Packages & Libraries

* NPM
* Three.js
* Cannon.js
* WebGL

3. Design Tools

* Blender
